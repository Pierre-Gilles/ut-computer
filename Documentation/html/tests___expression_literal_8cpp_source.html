55  /*
56  * Le résultat de l’opérateur binaire utilisant un symbole (+, -, /, *, $) appliqué entre la littérale expression
57  * ’EXP1’ et la littérale expression ’EXP2’ renvoie un littérale expression composée des deux littérales initiales,
58  * chacune entre parenthèses, les deux littérales étant séparées par le symbole.
59  * Par exemple, on obtient ’(EXP1 )+(EXP2)’ si l’opérateur est +. Les parenthèses autour des expressions
60  * EXP1 et/ou EXP2 sont omises dans le résultat si tous les opérateurs non parenthésés dans l’expression ont
61  * la même priorité que l’opérateur.
62  * Les opérateurs *, / ont une priorité supérieure à + et -.
63  * Les opérateurs *, / ont la même priorité entre eux.
64  * Les opérateurs + et - ont la même priorité entre eux.
65  * Les opérateurs numériques ont priorité sur les opérateurs logiques.
66  * Deux opérateurs qui ont la même priorité sont toujours évalués de la gauche vers la droite.
67  * Par exemple, la ligne de commande ’3+X’’9*Y’* a pour résultat ’(3+X)*9*Y’.
68  * */
